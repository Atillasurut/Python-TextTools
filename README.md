# Python Utilities - Beginner Version

A collection of simple Python utility functions for math, text, and password checks.  
This project introduces basic **OOP (classes)** in Python.

## Features
- Math operations: add, subtract, multiply, divide
- Number checks: is_even, is_odd
- List analysis: biggest, smallest
- Text tools: reverse text, palindrome check
- Password strength checker (weak/strong)

## How to Run
```bash
python text_tools.py
